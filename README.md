# nqcc
A simple C Compiler

This is a repo to hold my code to go with the [Writing a C Compiler](https://nostarch.com/writing-c-compiler) book by Nora Sandler.
The book has a [corresponding repo of tests](https://github.com/nlsandler/writing-a-c-compiler-tests), which are also to be used for testing.
The goal is to learn how to write a compiler for x86 from this book, and then transfer that over to [SlothPU16](https://github.com/freesurfer-rge/slothpu16).
